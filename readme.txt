== Description ==

Date Wise Page List - For generate a page list between two dates also generate a csv file of it.

* List pages between two dates and generate a csv file.

== Installation ==

1. Upload the directory `/between-date-page-list/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the plugin name on the plugins page.
